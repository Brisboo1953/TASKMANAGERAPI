Es una aplicación web que permite gestionar tareas.

Tecnologías utilizadas
-Node.js → Entorno de ejecución para JavaScript en el servidor
-Express.js → Framework para crear la API y manejar las rutas HTTP
-CORS → Middleware que permite la comunicación entre el frontend y el backend

Para el front end utilice 
Html y css
